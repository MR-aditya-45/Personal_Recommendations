# 📘 Personalized Learning Recommender

A **web-based intelligent system** that provides **personalized topic recommendations** and **study plans** for students based on their learning history and confidence levels. Students can filter topics by proficiency, create structured day-wise study plans, and download them as `.ICS` calendar files for integration with **Google Calendar** or **Outlook**.  

Built with **Python, Flask, Bootstrap, and Pandas**, this project aims to **optimize learning efficiency** and help students focus on areas that need improvement.

---

## 🚀 Features

- **Student Selection:** Choose a student from the landing page to view recommendations.  
- **Personalized Recommendations:** Topics are suggested based on prior performance and confidence levels.  
- **Confidence-Based Filtering:** Filter topics by Weak (<50%), Moderate (50–79%), or Strong (≥80%).  
- **Pagination:** Navigate recommendations easily with Previous/Next buttons.  
- **Study Plan Generation:** Create daily study plans with selected topics, hours per day, and start date.  
- **Downloadable Calendar:** Export the study plan as a `.ICS` file for Google Calendar, Outlook, etc.  
- **Resource Links:** Quick access to related videos and documents for each topic.  

---

## 🛠 Technology Stack

- **Backend:** Python, Flask  
- **Frontend:** HTML, CSS, Bootstrap 5  
- **Data Management:** Pandas, CSV files  
- **Calendar Export:** ics Python library  
- **Templating:** Jinja2
